# Odin Recipes

This project is part of the Foundations course from **The Odin Project**. In this project, I built a basic recipe website using only HTML and applied the concepts introduced throughout the HTML section of the curriculum.

By completing this project, I demonstrated an understanding of core HTML concepts, including creating and structuring web pages, using headings and paragraphs, creating ordered and unordered lists, adding links between pages, displaying images, and organizing files and directories within a multi-page website.

This project marked my experience building a complete website from scratch and provided a solid foundation for future learning in CSS, JavaScript, and full stack web development.
